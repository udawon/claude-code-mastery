# 개발자 웹 이력서 - 개발 로드맵

## 프로젝트 개요

순수 HTML, CSS, JavaScript, TailwindCSS를 활용한 개인 개발자 웹 이력서 사이트

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일링 | CSS3, TailwindCSS (CDN) |
| 인터랙션 | Vanilla JavaScript |
| 빌드 도구 | 없음 (정적 파일) |
| 배포 | GitHub Pages / Netlify |

---

## 섹션별 상세 내용

### 1. Header
- 이름/로고
- 네비게이션 메뉴 (각 섹션 앵커 링크)
- 스크롤 시 고정(sticky) 처리
- 모바일 햄버거 메뉴

### 2. Hero 섹션
- 이름: 홍길동
- 직함: Frontend Developer
- 한줄 소개: "사용자 경험을 최우선으로 생각하는 프론트엔드 개발자입니다."
- CTA 버튼: "이력서 다운로드" / "연락하기"
- 간단한 타이핑 애니메이션 효과

### 3. About (자기소개)
- 프로필 이미지 (플레이스홀더)
- 간단한 자기소개 (3~4줄)
- 경력 연차, 관심 분야 등 핵심 정보
- 예시 내용:
  > "3년차 프론트엔드 개발자로, 깔끔한 UI와 직관적인 UX를 만드는 데 열정을 가지고 있습니다.
  > 다양한 웹 프로젝트 경험을 통해 사용자 중심의 서비스를 개발하고 있습니다."

### 4. Skills (기술 스택)
- 카테고리별 분류:
  - **Frontend**: HTML, CSS, JavaScript, React, TailwindCSS
  - **Backend**: Node.js, Express
  - **Database**: MySQL, MongoDB
  - **Tools**: Git, GitHub, VS Code, Figma
- 프로그레스 바 또는 아이콘으로 시각화

### 5. Experience (경력 사항)
- 타임라인 형태 레이아웃
- 예시 경력:
  ```
  2023.03 ~ 현재    | (주)테크스타트 | 프론트엔드 개발자
                    | - 사내 관리자 대시보드 개발
                    | - 반응형 웹 서비스 유지보수
                    | - REST API 연동 및 성능 최적화

  2021.06 ~ 2023.02 | (주)웹솔루션 | 주니어 개발자
                    | - 기업 홈페이지 제작 및 운영
                    | - 크로스 브라우저 호환성 대응
                    | - 랜딩 페이지 퍼블리싱
  ```

### 6. Projects (프로젝트)
- 카드 형태 그리드 레이아웃
- 각 프로젝트 카드 구성:
  - 프로젝트 스크린샷 (플레이스홀더)
  - 프로젝트명
  - 간단한 설명
  - 사용 기술 태그
  - GitHub / Live Demo 링크 버튼
- 예시 프로젝트 3개:
  1. **Todo 앱** - React 기반 할일 관리 앱
  2. **날씨 대시보드** - OpenAPI 활용 날씨 정보 앱
  3. **포트폴리오 사이트** - 개인 이력서 웹사이트

### 7. Education (학력)
- 학교명, 전공, 졸업연도
- 관련 자격증/수료 내역
- 예시:
  ```
  2017.03 ~ 2021.02 | OO대학교 | 컴퓨터공학과 졸업
  2020.11           | 정보처리기사 취득
  2021.03           | OO 부트캠프 프론트엔드 과정 수료
  ```

### 8. Contact (연락처)
- 이메일
- GitHub 링크
- LinkedIn (선택)
- 블로그 (선택)
- 간단한 연락 폼 (이름, 이메일, 메시지) - UI만 구현

### 9. Footer
- 저작권 표시
- 맨 위로 스크롤 버튼

## 디자인 가이드

### 컬러 팔레트
```
Primary    : #2563EB (파란색 계열)
Secondary  : #1E293B (다크 네이비)
Accent     : #06B6D4 (시안)
Background : #F8FAFC (밝은 회색)
Text       : #334155 (진한 회색)
White      : #FFFFFF
```

### 타이포그래피
- 제목: Bold, 큰 사이즈
- 본문: Regular, 가독성 우선
- 웹폰트: Pretendard 또는 Noto Sans KR (CDN)

### 반응형 브레이크포인트
- 모바일: ~640px
- 태블릿: 641px ~ 1024px
- 데스크탑: 1025px~

## 개발 단계

### Phase 1: 기본 구조 (HTML)
- [ ] 프로젝트 폴더 구조 생성
- [ ] index.html 기본 마크업 작성
- [ ] 모든 섹션 시맨틱 태그로 구성
- [ ] TailwindCSS CDN 연결
- [ ] 웹폰트 연결

### Phase 2: 스타일링 (CSS + TailwindCSS)
- [ ] TailwindCSS 유틸리티 클래스로 레이아웃 구성
- [ ] 각 섹션 스타일 적용
- [ ] 반응형 디자인 처리
- [ ] 커스텀 CSS 애니메이션 추가 (styles.css)
- [ ] 다크모드 지원 (선택)

### Phase 3: 인터랙션 (JavaScript)
- [ ] 모바일 햄버거 메뉴 토글
- [ ] 스무스 스크롤 네비게이션
- [ ] 스크롤 시 섹션 페이드인 애니메이션
- [ ] Hero 섹션 타이핑 효과
- [ ] 맨 위로 스크롤 버튼

### Phase 4: 마무리
- [ ] 크로스 브라우저 테스트
- [ ] 모바일 디바이스 테스트
- [ ] 성능 최적화 (이미지 등)
- [ ] GitHub Pages 또는 Netlify 배포

## 참고 사항

- 빌드 도구 없이 정적 파일로만 구성
- TailwindCSS는 CDN 방식으로 사용
- 모든 UI 텍스트는 한국어로 작성
- SEO를 위한 meta 태그 포함
- Open Graph 태그로 링크 공유 시 미리보기 지원
